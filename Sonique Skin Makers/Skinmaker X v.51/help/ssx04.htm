INTRODUCTION TO SSX

What's the difference between the old skinmaker and SSX? Almost everything. What you'll notice first is the GUI which is more like Sonique. Skinmaker loads paths and commands and then compiles a skin for you. With SSX the skin is compiled im RAM as you load files into it or modify the settings. This means that you can open already saved/finished skins, modify them and then save them again.


A TOUR OF THE SSX INTERFACE

At first you see two buttons. To start making a skin, press the left one.
You are now presented with the info editing screen. That is the gray box on the bottom. Above that is a thinner box which is the editing screen selector. Here you have one page for each step in the skin creation process - Info, Images, Regions, Colors and Advanced.

Up to the right is the main menu where you find load, save, new, help, test and submit.

THE MAIN MENU (ON ALL PAGES)

The Load, Save and New buttons are to be used on skin files only.
If you hit the test button (after your skin is saved) windows does a "open" like when you double click the file in explorer.
When you are finished, hit the submit button. The program will then upload the skin to ftp.sonique.com and then the Sonique Skin engineer will upload it to the Skin site in the order in which it was recieved. Skins uploads are also judged based upon style, if a skin looks really nice it will be uploaded before one that looks poor.


THE INFO PAGE

Here you enter information about yourself and your skin. The three yellow fields are required.
The program will not tell you that and the risk you take is that your skin might not be submitted.

In the Author Name field, you type in your name.

In the Skin title field, you type in the name of the skin that you wish to appear in Sonique and on skins.sonique.com

In the Private e-mail field, you type in your private e-mail adress. This is only used for communications with you if there's a problem with the skin or something is uncertain.

In the Skin URL field, you type the full URL that is presented in Sonique when showing info about the skin. It's suggested you use http://mydomain/mypage or mailto:myemail@mydomain

In the URL Description field, you type in the text you want to appear as link text for the URL above.

In the Public E-mail field, you type in your e-mail that is going to be published in skins.sonique.com.

In the Description field, you type in a description that will be viewed by the skins.sonique.com staff only. You can tell them it's an update or whatever.

Below, you have two buttons that lets you load a skin thumbnail and a mask. Both in JPEG format and of the same size. The mask must be a BW JPEG. Black is transparent and white is solid.
Tip! View the loaded images by right-clicking on the buttons.

THE IMAGES PAGE

Here you load images and masks to use in your skin and to build regions from. Please note the formats of the images.
Here you can also right-click to bring up the Image Viewer.

THE REGIONS PAGE

Here you see all the regions listed in their respective mode. To bring up the region editor click any of them. For the region editor to work, it requires that you have loaded both the texture JPEG and the mask BMP. (Test files are available in skins/examples).

Here is what the region editor looks like:

On the top left you can select which region to edit and the viewing mode.
You can also select this with the arrow keys on your keyboard.
Further right, the Invert button inverts the region.
The clear button creates an empty region which can be used to remove buttons from Sonique.
When creating a region you can use two methods. Floodfill and ColorKey. Floodfill works pretty much like the paint bucket in photoshop while colorkey selects all pixels with that color (in the mask).
If you want to combine regions and create "Inverted Floodfill" and "Inverted ColorKey" you can do so by using Alt, Ctrl and Shift. The Three indicators on the bottom right show you which function you are selecting when you hold down Alt, Ctrl, and shift. (They change color)

THE COLORS PAGE

Here you edit the Sonique Colors. To get information about a color, click it's name (the turqouse text-buttons). Otherwise there's four editing tools available for each color.

  • The Three sliders (R,G,B)
  • The Windows standard color selection box (left click once on the color disp. box)
  • Start dragging in the color select box and suck colors right off your desktop. Useful in combination with a image viewing window open.
  • If you right-click the color selection box you can change the intensity of the color. (Simple add/sub)

THE ADVANCED PAGE

Here you only have one selection at the moment. (Will change in future versions)
The INI editor should be straight-forward but the contents are not. The comments in the default ini file (older skins usually lack such comments) are quite descriptive in most parts. If anything is unclear check these sources and if that doesn't help, e-mail ssxsupport@sonique.com.
