SONIQUE SKINMAKER X HELP SYSTEM - SKINMAKER X INSTALLATION & FILES


INTRODUCTION

Sonique Skinmaker X (hereafter referred to as SSX) is a third party application made to expand the Sonique skinmaking community by helping skin authors make skins with a new Sonique-like interface and hopefully better support files. The support files include the program's INI file, this help system and via e-mail. This is our way to contribute to the best audioplayer there is - Sonique.


SO WHAT HAPPENS WHEN I INSTALL SSX?

The installation program puts the SSX files in your Sonique directory. SSX is best kept in the Sonique directory where most of the action takes place. If, for some reason, you feel that you need to put SSX in a directory other than your Sonique directory, it is possible to move the program. In that case you need to create a dummy file called sonique.exe in the folder with SSX (this is the only check the program does to determine wether it's positioned in the Sonique directory or not). Moving SSX shouldn't affect the testing of skins since those events are initiated via Windows file-type system, but it is recommended that you keep the SSX files in your Sonique directory.

Installed files

    {your sonique directory} \   SSX.EXE - the SSX executable
    A couple of help html files beginning with [ssx] into {your sonique directory} \help
    The bitmaps used in this documentation into {your sonique directory} \images
    Also the program creates the directory: {your sonique directory} \skins\examples

Authors are welcome to contribute to future example packs by contacting ssxsupport@sonique.com.
